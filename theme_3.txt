<b>Умовні оператори</b>

<b>Опис</b>

Сьогодні на занятті ми отримаємо знання стосовно використання умовних операторів, створення складних умов з використанням булевої алгебри.

Сьогодні ми:

Розглянемо оператор вибору if…else.
Ознайомимось із конструкцією if ... elif ... else.
Ознайомимось з тернарним оператором.
Дізнаємось про булеву алгебру.
Розглянемо блоки інструкцій.
Виконаємо практичні завдання.

<b>Порядок виконання інструкцій програми</b>

За замовчуванням в інструкції Python виконуються одна за одною зверху вниз.

<b>Приклад:</b>

name = input("What is your name? ")
print(f"Hello {name}")

У прикладі дві інструкції спочатку виконається name = input("What is your name? "), потім print(f"Hello {name}").

Послідовність виконання виразів у програмі називається "потік виконання" (Flow of execution).

У Python існує три способи керування потоком виконання:

умовне виконання — виконання блоку інструкцій тільки, коли наступає певна умова;
цикли — повторення виконання блоку інструкцій поки виконується деяка умова;
винятки — виконання блоку інструкцій у разі помилки.

Умовне виконання

Приклад:

age = input("How old are you? ")
if int(age) >= 18:
    print("You are adult already.")
else:
    print("You are infant yet.")

У Python реалізовано оператора контролю виконання (умовний оператор) if ... elif ... else.

Оператор контролю виконання дозволяє виконувати блоки інструкцій який не завжди, а лише тоді, коли буде виконано умова.

Завдання_1

Ввести число з клавіатури і визначити, до якого типу воно належить: додатнє, від’ємне чи нуль.

Код

a = input('Введіть число')
a = int(a)
if a > 0:
	print('Число додатнє')
elif a < 0:
	print("Число від'ємне")
else:
 	print('Це число — нуль')

Пояснення:

Під час виконання умовного оператора інтерпретатор Python перевіряє умови згори донизу доти, доки не знайде ту, яка виконується.
Потім виконає вираз для цієї умови і вийде з перевірки умов.

Завдання_2

Перегляньте код, поданий нижче. Поясніть чому умова a == 1 ніколи не буде виконана.

<b>Код</b>

a = input('Введіть число')
a = int(a)
if a > 0:
    print('Число додатнє')

    print('Число доівнює 1')
