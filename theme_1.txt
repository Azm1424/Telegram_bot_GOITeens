<b>Що таке Python?</b>

<b>Python</b> — високорівнева, універсальна, інтерпретована, динамічна мова програмування.
ЇЇ переваги у читабельності коду та синтаксисі, який дозволяє програмістам висловлювати концепції меншою кількістю рядків коду, ніж це можливо на таких мовах, як C++ або Java.

Мова Python розробив голландський програміст <b>Гвідо Ван Россум (Guido van Rossum)</b> у 1991 році.
Гвідо був фанатом британського комедійного серіалу "Monty Python's Flying Circus", звідки і прийшла назва мови.

<b>Вихідний код (source code)</b> — це набір фраз, слів та спеціальних символів, специфічних для мови програмування, що описують набір інструкцій для комп'ютера

Але символи, слова та фрази, що становлять програму, насправді незрозумілі для машини. Є крок після написання програми, який конвертує вихідний код у файлі на набір інструкцій зрозумілих комп'ютеру.

Цим займається спеціальна програма: <b>компілятор</b> або <b>інтерпретатор</b>.

<b>Python онлайн-компілятори</b>

Для перевірки коду на Python підходить сервіс <b>Online Python (online-python.com)</b>

Тут представлена проста IDE, яка підтримує завантаження з комп'ютера та завантаження коду у вигляді файлу з розширенням *.py. Ви можете працювати над проєктом разом із друзями, поділившись посиланням.

У багатомовних компіляторах також дуже хороша підтримка Python.
